# jsTube API

Django JSON API for the JsTube media service.

This service stores display/search state in MongoDB and reads the original
image/video files through the webhard internal API. Source code only belongs in
this repository. Do not commit production `.env` files, tokens, database dumps,
media files, or private keys.

## Role

- Sync webhard image/video metadata into MongoDB.
- Serve media list, detail, album, favorite, like, delete, and public flags.
- Proxy media playback from webhard, including byte-range MP4 streaming and HLS.
- Create YouTube import and time-tag jobs for `jstube-worker`.
- Expose worker heartbeat/status data for the admin UI.
- Provide karaoke TV, remote, queue, pairing, and playback APIs.

## Streaming

Video items include both `content_url` and, when available, `hls_url`.

- `content_url` proxies the webhard file stream and preserves `Range` headers, so
  browsers receive `206 Partial Content` for seekable MP4 playback.
- `hls_url` proxies webhard HLS playlists and `.ts` segments. HLS files are
  generated by `jsDrive` transcode jobs and served through this API so browser
  cookies and media permissions stay consistent.

If HLS is not ready, the frontend falls back to `content_url`.

## Runtime

Important runtime values are provided by `jsDeploy`:

- `MEDIA_SERVICE_SECRET_KEY`
- `MEDIA_SERVICE_ALLOWED_HOSTS`
- `MEDIA_SERVICE_CORS_ORIGINS`
- `ADMIN_SERVICE_BASE_URL`
- `MEDIA_FRONTEND_BASE_URL`
- `MEDIA_QR_BASE_URL`
- `WEBHARD_PUBLIC_BASE_URL`
- `WEBHARD_INTERNAL_BASE_URL`
- `MEDIA_INTERNAL_API_TOKEN`
- `WEBHARD_STORAGE_ROOT`
- `MEDIA_MONGO_URI`
- `MEDIA_MONGO_DATABASE`
- `MEDIA_SYNC_LIMIT`
- `MEDIA_WORKER_EXPECTED_REPLICAS`
- `MEDIA_WORKER_STALE_SECONDS`

Production startup rejects unsafe defaults such as debug mode, wildcard hosts,
localhost-only CORS, short/default internal tokens, and weak secret keys.

```sh
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python manage.py check
python manage.py test media_api
python manage.py runserver 0.0.0.0:8085
```
